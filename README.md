# Primeiro PR

Repositório criado para praticar o fluxo de Pull Request com o Claude Code.

## O que é este repositório?

Um espaço seguro para aprender como funciona o fluxo de contribuição no GitHub:
criar uma branch, fazer uma alteração e abrir um Pull Request.

## Fluxo utilizado

1. Criar uma branch a partir da `main`
2. Fazer uma melhoria (por menor que seja)
3. Commitar e fazer push
4. Abrir um Pull Request
5. Revisar e fazer merge

## Tecnologias

- Git
- GitHub CLI (`gh`)
- Claude Code
